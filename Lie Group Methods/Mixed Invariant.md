$$\dot{X} = V_r X + X V_l$$
$V_r$ is [[right invariant]] vector field
$V_l$ is [[left invariant]] vector field

If left multiply $\dot{X}$ by [[Pullback]] $X^{-1}$ then $X^{-1}\dot{X}\in \mathfrak{g}$
$$X^{-1}\dot{X} = X^{-1}V_{r}X + V_l = \Ad{X^{-1}}{V_r} + V_l$$

If right multiply $\dot{X}$ by [[Pullback]] $X^{-1}$ then $X^{-1}\dot{X}$
$$\dot{X}X^{-1} = V_{r} + \Ad{X}{V_l} = \Ad{X^{-1}}{V_r} + V_l$$
